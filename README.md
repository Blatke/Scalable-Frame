# Scalable Frame
A mod for HS2 / AIS Studio item.

See the video for demonstration: https://youtu.be/-VQprMQ_zxI

## Shader
Since v0.0.4, it has adopted the shader of [GlassBumpRefraction Shader](https://github.com/Blatke/GlassBumpRefraction-Shader-for-ME) to the glass walls in the among the frames. Please click the link and install this shader mod, this would let it be possible to modify refractive effect of the glasses via MaterialEditor tab.
